# Founder Dashboard Guide

## Purpose

This dashboard is meant to give complete visibility into the founder loop:

- daily productivity
- growth activity
- revenue progress
- newsletter momentum
- feedback capture
- launch blockers

## How To Use It Daily

### Morning

- Review the 8:00 AM scan
- Decide the top revenue move for the day
- Set outreach and content targets
- Check which blockers still need to be wired

### Midday

- Update outreach count
- Update replies and sales conversations
- Mark whether a content asset shipped
- Note any repeated pain points or objections

### End Of Day

- Update revenue collected
- Log one meaningful insight
- Decide tomorrow's highest-leverage move

## What Complete Visibility Means

The dashboard should answer these questions quickly:

- Did I do enough useful work today?
- Did I talk to the market enough?
- Did anything create or unblock revenue?
- Is the newsletter growing or stalled?
- What are people actually saying?
- What system is blocking the next level of growth?

## Suggested Manual Sources

- Payment dashboard
- Email or DM inbox
- Newsletter platform
- Feedback form responses
- Daily opportunity scan report
- Personal founder notes

## Weekly Review

At the end of each week, ask:

- Which actions created the most revenue?
- Which channel created the strongest response?
- Which message angle converted best?
- Which newsletter issue got the strongest engagement?
- Which repeated objection should change the product or offer?
