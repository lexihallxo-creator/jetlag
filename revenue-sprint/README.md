# JetLag Revenue Sprint

This folder contains the fastest path to first revenue for JetLag without waiting for the full app to ship.

## Goal

Sell a manual concierge version of JetLag in the next 24 hours.

## What Is In Here

- `24-hour-revenue-plan.md`: exact action plan for today
- `offer.md`: the paid offer, pricing, and positioning
- `outreach-scripts.md`: DM, comment, and follow-up scripts
- `client-intake-form.md`: what to ask after someone says yes
- `delivery-template.md`: how to fulfill the service quickly
- `landing-page/index.html`: simple landing page for the offer

## Recommended Flow

1. Read `24-hour-revenue-plan.md`
2. Publish or copy the landing page copy
3. Start outreach using `outreach-scripts.md`
4. Send the intake questions from `client-intake-form.md`
5. Deliver from `delivery-template.md`
