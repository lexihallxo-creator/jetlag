# PRD: Smart Itinerary Import

## Problem

Users should not need to manually rebuild their trip just to receive a jet lag plan.

## Goal

Let users create a usable trip record by pasting travel details or importing itinerary information with minimal editing.

## User Value

- Faster setup
- Less manual time-zone confusion
- Higher confidence that the plan matches the real trip

## Success Metrics

- Trip creation completion rate
- Share of trips created via import
- Time from signup to first generated plan
