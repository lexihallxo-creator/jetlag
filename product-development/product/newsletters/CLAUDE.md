# Newsletters

This folder contains JetLag's newsletter system.

## Purpose

Build an audience asset that is actually worth reading, not generic wellness content. The newsletter should create trust, repeat attention, and conversion opportunities by delivering distinctive travel-performance insight.

## What Lives Here

| Path | Purpose |
|------|---------|
| `newsletter-strategy.md` | Editorial positioning and differentiation |
| `format-library.md` | Repeatable issue formats that feel unique |
| `publishing-workflow.md` | How to source, draft, edit, and publish each issue |
| `content-calendar.md` | Near-term themes and publishing rhythm |
| `issue-template.md` | Reusable structure for writing issues |
| `issues/` | Drafted or published newsletter issues |

## Newsletter Standard

Every issue should do at least one of these:

- Teach something unusually useful
- Reframe a traveler pain point in a memorable way
- Give an immediately usable action plan
- Make the reader feel more prepared than they were 5 minutes earlier
