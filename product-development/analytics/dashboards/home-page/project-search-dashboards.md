# Project Search - Dashboards

Dashboards for monitoring and analyzing the Project Search (Cmd+K) feature.

| Dashboard | Tool | URL | Description | Owner |
|-----------|------|-----|-------------|-------|
| Search Health | Sigma | [link](https://app.sigmacomputing.com/forge-labs/workbook/search-health) | Real-time search usage rate, zero-results rate, latency percentiles, result distribution by type | Casey Nguyen, Analytics |
| Search Experiments | Amplitude | [link](https://analytics.amplitude.com/forge-labs/dashboard/search-experiments) | A/B test results for search ranking, UI variants, and feature flag rollout metrics | Casey Nguyen, Analytics |
| Search Funnel | Mode | [link](https://app.mode.com/forge-labs/reports/search-funnel) | End-to-end funnel from modal open to project engagement, segmented by tier and cohort | Hannah Stulberg, PM |
